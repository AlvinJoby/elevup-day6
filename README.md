## 🔑 TMDb API Setup

1. Create a free TMDb account.
2. Generate an API key.
3. Open `script.js`.
4. Replace:

```js
const API_KEY = "YOUR_API_KEY";
```

with:

```js
const API_KEY = "YOUR_TMDB_API_KEY";
```
